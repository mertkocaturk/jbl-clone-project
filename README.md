# 🎧 JBL Clone

Bu projeyi, HTML, SCSS ve JavaScript bilgimi geliştirmek amacıyla hazırladım. Gerçek bir ürün tanıtım sayfasını referans alarak responsive bir landing page oluşturdum. Projede modern tasarım anlayışını koruyarak animasyonlar, mobil menü ve kullanıcı deneyimini iyileştiren çeşitli arayüz detayları ekledim.

## 🚀 Proje Özellikleri

- Responsive (Mobil, Tablet ve Masaüstü uyumlu) tasarım
- Hamburger menü
- Modern ve şık landing page tasarımı
- CSS animasyonları ve geçiş efektleri
- SCSS kullanılarak modüler dosya yapısı
- Font Awesome ikonları
- Temiz ve okunabilir kod yapısı

## 🛠️ Kullanılan Teknolojiler

- HTML5
- SCSS (Sass)
- JavaScript (ES6)
- CSS Flexbox
- Font Awesome

## 📂 Proje Yapısı

```text
├── images
├── styles
│   ├── _about.scss
│   ├── _favorite.scss
│   ├── _footer.scss
│   ├── _header.scss
│   ├── _home.scss
│   ├── _variables.scss
│   └── style.scss
├── index.html
└── main.js
```

## 🎯 Bu Projede Kendimi Geliştirdiğim Konular

- SCSS dosyalarını modüler şekilde organize etme
- Responsive tasarım geliştirme
- JavaScript ile DOM manipülasyonu
- Hamburger menü oluşturma
- CSS animasyonları ve hover efektleri
- Daha düzenli ve sürdürülebilir kod yazma
